# User Management Plugin

## Template Admin
[AdminLTE](https://github.com/ColorlibHQ/AdminLTE)

Migration and Seeding table:
```sh
php artisan migrate:plugin
```

## Enable Plugin

```sh
composer dumpautoload
```

## Publish Plugin

```sh
php artisan vendor:publish
```
